🌿 GreenDen – A Clean & Minimal Plant Store UI
GreenDen is a beautifully designed front-end web project built with Tailwind CSS. It showcases a modern, responsive plant store layout with clean typography, soft colors, and elegant spacing — ideal for learning and demonstrating Tailwind in action.

✨ Features
🌱 Fully responsive layout

🎨 Tailwind CSS for utility-first styling

🧩 Clean and reusable UI components

📱 Mobile-first design

🌍 Beautiful plant shop theme
